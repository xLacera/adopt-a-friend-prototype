# Adopt a Friend 🐾

Plataforma de adopción de mascotas. Cada mascota merece un hogar lleno de amor.

## Tecnologías

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Instalación

```bash
npm install
npm run dev
```
